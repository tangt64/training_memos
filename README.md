## 메모파일 설명

- 이 저장소는 더 이상 메모는 없습니다. 모든 메모는 미로를 통해서 공유 및 제공되고 있습니다.
- PDF파일은 각 년도별로 구성이 되어 있습니다. 브렌치(branch)를 통해서 확인 해주시면 됩니다.
- 년도별로 PDF파일은 구성이 되어 있습니다. 내용은 최신 년도만 업데이트 됩니다.
- 강의 및 랩에서 사용하는 코드는 https://github.com/tangt64/codelab 에서 별도로 제공하고 있습니다. 더 이상 예제 코드는 같이 제공하지 않습니다.
- 모든 교육은 오픈스택 기반으로 구성이 되어 있습니다.
